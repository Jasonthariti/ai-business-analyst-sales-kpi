# Sales KPI Analysis

This is a simple sales analysis project focused on understanding sales performance using common business KPIs.
The project is designed to practice working with sales data and presenting results in a way that can support basic business decisions.

## Overview
The project uses transaction-level sales data and summarizes it into clear and useful KPIs.
It includes overall sales performance, product-level analysis, and monthly sales trends.

The code is organized in a modular structure so that data loading and KPI calculations are separated from the notebook.

## What This Project Includes
- Overall sales KPIs (total revenue, profit, profit margin)
- Product-level sales performance analysis
- Monthly sales trends and revenue growth
- Clean and reusable Python code structure

## Tools Used
- Python
- pandas
- Jupyter Notebook

## Notes
This project focuses on basic business analysis and KPI calculation.
It does not include machine learning or forecasting and is intended as a practical analytics example.
